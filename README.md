## Surge
仅 Surge iOS
```
surge:///start
启动当前选中的配置文件。
surge:///stop
停止当前会话。
surge:///toggle
启动或停止当前选中的配置文件。
surge:///install-config?url=x
从某个 URL 安装配置文件。该 URL 需先通过百分号编码处理。
参数：
autoclose=true
操作完成时自动关闭 Surge。（不能与 install-config 一同使用）
样例：surge:///toggle?autoclose=true
```
